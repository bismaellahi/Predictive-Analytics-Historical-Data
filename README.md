# Predictive Analytics Using Historical Data

## Objective
Build a predictive model to forecast future sales using historical data.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Features
- Data preprocessing
- Linear Regression model
- Future sales prediction
- Model evaluation (MSE & R²)
- Data visualization

## Results
- Mean Squared Error (MSE): 11.667
- R² Score: 0.9967
